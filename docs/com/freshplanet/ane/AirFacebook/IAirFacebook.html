dialog()method 
public function dialog(method:String, parameters:Object = null, callback:Function = null, allowNativeUI:Boolean = true):void

Open a Facebook dialog.

Parameters

method:String — A dialog method (eg. login, feed...).
 
parameters:Object (default = null) — (Optional) An object (key-value pairs) containing the dialog parameters.
 
callback:Function (default = null) — (Optional) A callback function of the following form: function myCallback(data:Object), where data is the parsed JSON object returned by Facebook.
 
allowNativeUI:Boolean (default = true) — (Optional) If true, we will try to use the native sharing sheet on iOS 6. Native sharing sheet will only be used if method is feed and parameters doesn't contain a non-empty to parameter. If the native sharing sheet can be used, only the following parameters will be used: name, picture, link. Default is true.
